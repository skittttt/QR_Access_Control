Мобильное приложение для контроля доступа на мероприятия по QR-коду. В проекте реализованы два режима работы: Пользователь получает QR-код для входа на событие, а Администратор сканирует код и принимает решение о входе.

Возможности

выбор роли (User / Administrator)

отображение списка мероприятий

просмотр подробной информации

генерация индивидуального QR-кода (JSON: userId, eventId, issuedAt)

сканирование QR-кода через CameraX + MLKit

принятие решения «Пропустить / Отказать»

учёт повторных входов

mock-имитация сервера

Технологии

MVVM

Navigation Component

Hilt (Dagger)

CameraX

MLKit

ZXing
